# Installation

This chapter covers all aspects of installing VirtRigaud in your Kubernetes cluster.

## Overview

VirtRigaud can be installed in several ways:
- Helm chart (recommended for production)
- kubectl apply with manifests
- GitOps with FluxCD or ArgoCD

Choose the installation method that best fits your infrastructure and operational model.
